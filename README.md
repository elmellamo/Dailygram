# Dailygram
Google의 Firebase를 사용한 Android SNS App

'Instagram'을 모방한 학습 목적 프로젝트


### 주요 기능
1. 회원 가입(가입 시 이메일 인증 기능 포함)
2. 로그인 / 로그아웃
3. 타임라인 페이지, 업로드 페이지, 회원 검색 페이지, 내 프로필 페이지 구현
4. 팔로워들의 게시물에 좋아요, 댓글
5. 게시물 업로드(갤러리, 카메라 접근)
6. 다른 회원 검색, 팔로우
7. 내 프로필 편집(프로필 사진, 소개 멘트 등)
